# GeekPark-mac

## 简介

一个基于swift语言和Mac原生控件开发的极客公园第三方客户端。

<img src="resources/screenshot.png" width="370">


## 编译和运行

### 编译环境

v1.0版本: `macos10.12.5`/`Xcode7.3.1`/`Swift2.2`

### 运行

 	git clone https://github.com/tzshlyt/GeekPark-mac
	cd GeekPark
    pod install
    open GeekPark.xcworkspace
